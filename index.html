Overview of the Web

The Web is composed of multiple HTML (Hypertext Markup Language) pages. The major pieces of the Web are you, your computer and browser, HTTP (protocol that browsers use to connect to servers), the internet (which uses HTTP to communicate with you), your browser, and servers.

HTML

HTML, or HyperText Markup Language, is the heart of the Web and is made up of text/content (what you see), markup tags (what it looks like), references to other documents (e.g., images and videos, and links to other pages, such as Google.com.

HTML documents have many components, but what was focused on in this lesson was primarily the body/contents of an HTML document. In the lesson, I was introduced to a number of tags and learned the difference between inline and block elements. Block elements form an invisible box around text that you can then, for example, assign a height and width to. I thought this was pretty cool.

What Computers Know

Computers are often called stupid. What this means is that a computer needs very specific instructions from a programmer in order to know how to perform. The computer only knows how to do exactly what you tell it to do.

Good news--programmers can't remember everything and aren't expected to!

This was the most encouraging part of the lesson, i.e., programmers can't remember everything. It was good to know that there are good resources available to help you remember commond HTML syntax.
